üîß Running the Examples

All examples are designed to be self-contained and runnable. Here's how to get started:

1. Clone the Repository

git clone https://github.com/elevationai/CORBA-ts.git cd CORBA-ts/examples

2. Run an Example

# Start the server in one terminal deno run --allow-net --allow-read --allow-write hello-world-server.ts # Run the client in another terminal deno run --allow-net --allow-read hello-world-client.ts

3. Explore and Modify

üêõ Troubleshooting

Common Issues

  • Permission denied - Make sure to include required --allow-net flags
  • Connection refused - Ensure the server is running before starting the client
  • Type errors - Check that you're using the correct interface types
  • Module not found - Verify your import paths and JSR package names
