Hello World Example

This is the simplest possible CORBA.ts application - a "Hello World" server that responds to client requests. It demonstrates the fundamental concepts of CORBA distributed computing.

üñ•Ô∏è Server Implementation

The server creates a servant that implements a simple greeting service:

// hello-server.ts import { init, ORB_instance, getRootPOA } from "jsr:@eai/corba"; import { Servant } from "jsr:@eai/corba/poa"; /** * HelloServant implements the greeting service. * Every servant must extend the Servant base class. */ class HelloServant extends Servant { /** * Repository ID identifies the interface this servant implements. * This follows the IDL repository ID format. */ _repository_id(): string { return "IDL:Hello:1.0"; } /** * Business method - creates personalized greeting * @param name - Name of the person to greet * @returns Personalized greeting message */ sayHello(name: string): string { const timestamp = new Date().toISOString(); console.log(`[${timestamp}] Server: sayHello called with name: "${name}"`); return `Hello, ${name}! Welcome to CORBA.ts üëã`; } /** * Utility method - returns current server time * @returns Current server time as ISO string */ getServerTime(): string { const now = new Date(); console.log(`[${now.toISOString()}] Server: getServerTime called`); return now.toISOString(); } } /** * Main server function */ async function startHelloServer() { try { console.log("üöÄ Starting Hello World CORBA.ts server..."); // Step 1: Initialize the CORBA runtime await init(); console.log("‚úÖ CORBA runtime initialized"); // Step 2: Get the ORB instance const orb = ORB_instance(); console.log("‚úÖ ORB instance obtained"); // Step 3: Get the Root POA and its manager const rootPOA = getRootPOA(); const poaManager = rootPOA.the_POAManager(); console.log("‚úÖ Root POA obtained"); // Step 4: Activate the POA Manager to start accepting requests await poaManager.activate(); console.log("‚úÖ POA Manager activated"); // Step 5: Create the servant instance const servant = new HelloServant(); console.log("‚úÖ HelloServant created"); // Step 6: Activate the servant in the POA const objectId = await rootPOA.activate_object(servant); console.log(`‚úÖ Servant activated with Object ID: ${objectId}`); // Step 7: Get the object reference for the activated servant const helloRef = await rootPOA.id_to_reference(objectId); console.log("‚úÖ Object reference obtained"); // Step 8: Convert object reference to IOR string const helloIOR = await orb.object_to_string(helloRef); // Step 9: Display the IOR for client use console.log("\n" + "=".repeat(60)); console.log("üìã SERVER READY!"); console.log("üìã Copy this IOR for your client:"); console.log("=".repeat(60)); console.log(helloIOR); console.log("=".repeat(60)); // Step 10: Save IOR to file for easy client access try { await Deno.writeTextFile("hello.ior", helloIOR); console.log("üíæ IOR saved to 'hello.ior' file"); } catch (error) { console.warn("‚ö†Ô∏è Could not save IOR to file:", error.message); } console.log("\nüèÉ Server is running and ready to process requests..."); console.log("üìû Waiting for client connections..."); console.log("üõë Press Ctrl+C to shutdown the server"); // Step 11: Start the ORB event loop (this will block) await orb.run(); } catch (error) { console.error("‚ùå Server startup failed:", error); if (error.stack) { console.error("Stack trace:", error.stack); } Deno.exit(1); } } /** * Handle graceful shutdown */ function setupShutdownHandler() { Deno.addSignalListener("SIGINT", async () => { console.log("\nüõë Shutdown signal received..."); console.log("üßπ Cleaning up resources..."); try { const orb = ORB_instance(); await orb.shutdown(true); // Wait for active requests to complete console.log("‚úÖ ORB shutdown complete"); } catch (error) { console.error("‚ö†Ô∏è Error during shutdown:", error.message); } console.log("üëã Hello Server stopped. Goodbye!"); Deno.exit(0); }); } // Main entry point if (import.meta.main) { setupShutdownHandler(); await startHelloServer(); }

üíª Client Implementation

The client connects to the server and makes remote method calls:

// hello-client.ts import { init, ORB_instance } from "jsr:@eai/corba"; /** * TypeScript interface defining the Hello service contract. * This provides type safety for remote method calls. */ interface HelloService { sayHello(name: string): Promise<string>; getServerTime(): Promise<string>; } /** * Main client function */ async function runHelloClient() { try { console.log("üîå Starting Hello World CORBA.ts client..."); // Step 1: Initialize the CORBA runtime await init(); console.log("‚úÖ CORBA runtime initialized"); // Step 2: Get the ORB instance const orb = ORB_instance(); console.log("‚úÖ ORB instance obtained"); // Step 3: Read the IOR from file (created by the server) let iorString: string; try { iorString = await Deno.readTextFile("hello.ior"); console.log("‚úÖ IOR loaded from 'hello.ior' file"); } catch (error) { console.error("‚ùå Could not read 'hello.ior' file"); console.error(" üí° Make sure the server is running and has created the IOR file"); console.error(" Error details:", error.message); Deno.exit(1); } // Step 4: Convert IOR string to object reference const helloRef = await orb.string_to_object(iorString); console.log("‚úÖ Object reference obtained from IOR"); // Step 5: Cast to our HelloService interface for type safety const helloService = helloRef as unknown as HelloService; console.log("\nüìû Making remote calls to the server..."); // Step 6: Make remote method calls try { // Call 1: Simple greeting const greeting = await helloService.sayHello("CORBA.ts Developer"); console.log("üì• Server response:", greeting); // Call 2: Get server time const serverTime = await helloService.getServerTime(); console.log("üì• Server time:", serverTime); // Step 7: Demonstrate multiple calls console.log("\nüîÑ Making multiple calls to show persistent connection..."); const names = ["Alice", "Bob", "Charlie"]; for (let i = 0; i < names.length; i++) { const name = names[i]; console.log(`\nüì§ Call ${i + 1}: Greeting ${name}...`); const response = await helloService.sayHello(name); console.log(`üì• Response ${i + 1}:`, response); // Small delay between calls await new Promise(resolve => setTimeout(resolve, 500)); } // Step 8: Final server time check console.log("\nüì§ Final call: Getting updated server time..."); const finalTime = await helloService.getServerTime(); console.log("üì• Final server time:", finalTime); console.log("\n‚úÖ All remote calls completed successfully!"); console.log("üéâ Hello World client finished!"); } catch (error) { console.error("\n‚ùå Error during remote method call:", error.message); // Provide helpful error messages if (error.message?.includes("Connection refused")) { console.error(" üí° The server appears to be down. Please start the server first."); } else if (error.message?.includes("timeout")) { console.error(" üí° The server is not responding. Check network connectivity."); } else { console.error(" üí° This might be a CORBA protocol error or server issue."); } throw error; } } catch (error) { console.error("\n‚ùå Client failed:", error); if (error.stack) { console.error("Stack trace:", error.stack); } Deno.exit(1); } } // Main entry point if (import.meta.main) { await runHelloClient(); }

üìö Code Explanation

Server-Side Components

Servant Class

The HelloServant class extends Servant and implements the business logic:

  • _repository_id() - Identifies the interface (required for all servants)
  • sayHello() - Business method that generates personalized greetings
  • getServerTime() - Utility method returning current server time

Object Lifecycle

  1. Create - Instantiate the servant class
  2. Activate - Register with POA using activate_object()
  3. Reference - Get object reference with id_to_reference()
  4. Export - Convert to IOR string with object_to_string()

Client-Side Components

Object Reference Resolution

The client follows this pattern to access remote objects:

  • Load IOR - Read the IOR string from file or configuration
  • Convert - Use string_to_object() to get object reference
  • Cast - Type-cast to the expected interface for type safety
  • Invoke - Make remote method calls using await

Network Communication

CORBA.ts handles all the network complexity automatically:

üîÑ Marshalling

Parameters and return values are automatically converted to/from network format (CDR)

üåê GIOP/IIOP

Uses standard CORBA protocols for interoperability with other CORBA implementations

‚ö° Async Operations

All remote calls return Promises, integrating seamlessly with modern JavaScript

üîí Type Safety

TypeScript interfaces provide compile-time type checking for remote methods
